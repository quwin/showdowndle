# Pokémon Showdowndle - An Unoffical Wordle-Like Game Based off Pokémon Showdown Usage Data
A simple little game where you can pick a format and keep guessing until you find the randomly Pokémon from in that format.
Available at https://showdowndle.vercel.app/.

Huge thanks to the people at Smogon and Pokémon Showdown for making all their code and data so readily available, and also big thanks to PokeAPI for base stat and type data.
